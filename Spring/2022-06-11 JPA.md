## JPA

Java 진영의 ORM 기술 표준으로 데이터베이스를 객체에 매핑해서 사용할 수 있게 해준다.

### 라이브러리 설치

#### Gradle

```
implementation 'org.springframework.boot:spring-boot-starter-data-jpa'
```

### 테이블 자동 생성 기능 끄기

application.properties 파일 안에 다음의 내용을 추가해주면 된다.

```
spring.jpa.hibernate.ddl-auto=none
```

### SQL 쿼리문 출력

application.properties 파일 안에 다음의 내용을 추가해주면 된다.

```
spring.jpa.show-sql=true
```

### 예제

#### Member (Entity)

``` java
@Entity
public class Member {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    @Column
    private String name;

    public Long getId() {
        return id;
    }

    public void setId(Long id) {
        this.id = id;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

}
```

#### MemberRepository (Repository)

``` java
@Repository
public class MemberRepository {

    private final EntityManager entityManager;

    @Autowired
    public MemberRepository(EntityManager entityManager) {
        this.entityManager = entityManager;
    }

    public Member save(Member member) {
        entityManager.persist(member);

        return member;
    }

    public Optional<Member> findById(Long id) {
        Member member = entityManager.find(Member.class, id);

        return Optional.ofNullable(member);
    }

    public Optional<Member> findByName(String name) {
        // Primary Key를 대상으로 하는 동작이 아닐 경우, JPQL로 쿼리문을 직접 작성
        List<Member> result = entityManager.createQuery("SELECT m FROM Member m WHERE m.name = :name", Member.class)
                .setParameter("name", name)
                .getResultList();

        return result.stream().findAny();
    }

    public List<Memeber> findAll() {
        // Primary Key를 대상으로 하는 동작이 아닐 경우, JPQL로 쿼리문을 직접 작성
        return entityManager.createQuery("SELECT m FROM Member m", Member.class)
                .getResultList();
    }

}
```

#### MemberService (Service)

``` java
@Service
@Transactional
public class MemberService { ... }
```
