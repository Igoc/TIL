## 의존성 자동 주입 대상이 없을 때 해결 방법

@Autowired의 required 옵션은 기본적으로 true로 되어 있어, 의존성 자동 주입 대상이 없으면 오류가 발생한다.

### @Autowired(required = false)

- 메서드 레벨에 붙어있다면 의존성 자동 주입 대상이 없을 때 메서드 자체가 호출되지 않는다.
- 필드 레벨에 붙어있다면 의존성 자동 주입 대상이 없을 때 null로 초기화된 상태로 남아있는다.

### org.springframework.lang.Nullable 애노테이션

의존성 자동 주입 대상이 없으면 null이 입력된다.

### Optional

의존성 자동 주입 대상이 없으면 비어있는 Optional이 입력된다.
