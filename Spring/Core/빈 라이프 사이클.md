## 빈 라이프 사이클

빈 생성 → 의존성 주입 → 초기화 콜백 → 사용 → 소멸 콜백 → 빈 소멸

### 예외 사항

- 생성자 주입의 경우 빈 생성과 의존성 주입이 동시에 일어난다.
- 프로토타입 빈의 경우 스프링 컨테이너에서 초기화 콜백까지만 관여한다.
