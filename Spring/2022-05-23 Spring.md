## Spring

Java 플랫폼을 위한 애플리케이션 프레임워크로, 웹 애플리케이션을 구축하기 위한 기능을 제공한다.

### Controller

클라이언트의 요청을 처리하기 위해 필요한 구성 요소를 호출하고, 처리한 결과를 다시 클라이언트에게 반환해주는 역할을 수행한다.

``` java
@Controller
class ExampleController {

    // index.html에 정의된 View를 반환하는 GET 방식의 라우터 정의
    @GetMapping("/")
    public String index() {
        return "index";
    }

    // HTTP 메시지의 Body에 직접 데이터를 반환하는 GET 방식의 라우터 정의
    @GetMapping("/greeting")
    @ResponseBody
    public String getGreeting(@RequestParam("name") String name) {
        return "Hello, " + name + "!";
    }

}
```

#### @RestController

`@Controller` 어노테이션과 `@ResponseBody` 어노테이션이 결합된 것으로, 하위 메서드에 `@ResponseBody` 어노테이션을 붙이지 않아도 문자열이나 JSON 등의 형식으로 응답할 수 있다.

``` java
@RestController
class MemberController {

    private final MemberService memberService;

    @Autowired
    public MemberController(MemberService memberService) {
        this.memberService = memberService;
    }

    // POST 방식의 라우터 정의
    @PostMapping("/member")
    public Long createMember(@RequestBody MemberDto memberDto) {
        return memberService.join(new Member(memberDto));
    }

    // GET 방식의 라우터 정의
    @GetMapping("/member/{name}")
    public String concatenateNameWithMsg(@PathVariable String name, @RequestParam("msg") String msg) {
        return name + " " + msg;
    }

}
```

### Service

비즈니스 로직을 수행하는 부분으로, Repository가 데이터베이스로부터 가져온 데이터를 전달받아 가공하는 역할을 수행한다.

``` java
@Service
class MemberService {

    private final MemberRepository memberRepository;

    @Autowired
    public MemberService(MemberRepository memberRepository) {
        this.memberRepository = memberRepository;
    }

    // 회원 가입 서비스
    public Long join(Member member) {
        memberRepository.save(member);

        return member.getId();
    }

}
```

### Repository

데이터베이스에 접근하여 데이터를 처리하는 역할을 수행한다.

``` java
class MemoryMemberRepository {

    private static Map<Long, Member> database = new HashMap<>();
    private static Long sequence;

    public Member save(Member member) {
        member.setId(++sequence);
        database.put(member.getId(), member);

        return member;
    }

}
```

### DTO

계층 간에 데이터를 교환할 때 사용하기 위한 용도로 정의하는 객체이다.

``` java
class MemberDto {

    private String name;

    public MemberDto() {}

    public MemberDto(String name) {
        this.name = name;
    }

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

}
```
