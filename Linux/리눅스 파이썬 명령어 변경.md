## python 명령어를 python3 명령어로 대치

``` bash
update-alternatives --install /usr/bin/python python /usr/bin/python3 1
```

## pip 명령어를 pip3 명령어로 대치

``` bash
update-alternatives --install /usr/bin/pip pip /usr/bin/pip3 1
```
