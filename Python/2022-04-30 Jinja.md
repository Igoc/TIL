## Jinja

Python에서 HTML 파일에 동적으로 데이터를 삽입할 수 있도록 만들어주는 템플릿 엔진이다.

### 패키지 설치

``` bash
pip install Jinja2
```

### 변수 생성

`{% set <변수명> = <값> %}` 형식을 이용하여 변수를 생성할 수 있다.

``` html
<!DOCTYPE html>

<html>
    <head>
    </head>
    <body>
        {% set name = "Alice" %}
    </body>
</html>
```

### 표현식 출력

`{{ ... }}` 형식을 이용하여 변수와 같은 표현식을 출력할 수 있다.

``` html
<!DOCTYPE html>

<html>
    <head>
        <script>
            let status = '{{ status }}';
        </script>
    </head>
    <body>
        <h3>Hello, {{ name }}!</h3>
    </body>
</html>
```

### 반복문

`{% for <요소를 담을 변수> in <반복을 수행할 변수> %} ... {% endfor %}` 형식을 이용하여 반복을 수행할 수 있다.

``` html
<!DOCTYPE html>

<html>
    <head>
    </head>
    <body>
        {% for name in names %}
            <h3>{{ name }}</h3>
        {% endfor %}
    </body>
</html>
```

### 조건문

`{% if <조건> %} ... {% elif <조건> %} ... {% else %} ... {% endif %}` 형식을 이용하여 조건에 따라 분기할 수 있다.

``` html
<!DOCTYPE html>

<html>
    <head>
    </head>
    <body>
        {% if status == "new" %}
            <button>회원가입</button>
        {% elif status == "old" %}
            <button>회원 탈퇴</button>
        {% else %}
            <div>에러</div>
        {% endif %}
    </body>
</html>
```

### 필터

변수의 값을 조작하기 위한 기능으로 `|`를 변수 뒤에 붙임으로써 사용할 수 있다.

#### safe

HTML 태그 등이 이스케이프 되지 않도록 만들어준다.

``` html
<!DOCTYPE html>

<html>
    <head>
    </head>
    <body>
        <h3>{{ example | safe }}</h3>
    </body>
</html>
```

#### tojson

객체를 JSON 문자열로 직렬화한다.

``` html
<!DOCTYPE html>

<html>
    <head>
        <script>
            let data = {{ data | tojson }};
        </script>
    </head>
    <body>
    </body>
</html>
```
