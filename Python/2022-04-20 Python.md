### f-string

문자열에 Python 표현식의 값을 삽입할 수 있는 문자열 포맷팅 방식이다.

``` python
name = '홍길동'
text = f'내 이름은 {name}!'

print(text)
```

``` bash
[출력]
내 이름은 홍길동!
```

#### 리스트(List)

순서를 고려하여 데이터를 가지고 있는 자료형이다.

``` python
# 리스트 선언
data = [1, 2, 'Alice', 3]

print(data[1])
print(data[2])

# 리스트에 새로운 요소 추가
data.append('Bob')

print(data)
```

``` bash
[출력]
2
Alice
[1, 2, 'Alice', 3, 'Bob']
```

#### 딕셔너리(Dictionary)

키(Key)-값(Value) 쌍으로 구성된 데이터를 가지고 있는 자료형이다.

``` python
# 딕셔너리 선언
user = {
    'name': 'Alice',
    'age': 21
}

print(user['name'])
print(user['age'])

# 딕셔너리에 새로운 키-값 추가
user['height'] = 160

print(user)
```

``` bash
[출력]
Alice
21
{'name': 'Alice', 'age': 21, 'height': 160}
```

### 함수

``` python
# 함수 선언
def sum(num1, num2):
    return num1 + num2

# 함수 호출
sum(3, 5)
```

### 람다(Lambda)

하나의 표현식으로 나타내어지는 익명 함수이다.

``` python
# 람다 선언과 동시에 호출
(lambda num1, num2: num1 + num2)(3, 5)

# 람다를 변수에 할당하여 사용
sum = lambda num1, num2: num1 + num2

sum(3, 5)
```

### datetime 패키지를 이용한 날짜 다루기

``` python
from datetime import datetime

# 현재 날짜 획득
date = datetime.now()

print(date)

# 날짜를 특정 포맷에 맞게 변환
formatted_date = datetime.now().strftime('%Y-%m-%d %H-%M-%S')

print(formatted_date)
```

``` bash
[출력]
2022-04-28 22:47:52.124803
2022-04-28 22-50-26
```
