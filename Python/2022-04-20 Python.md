#### 리스트(List)

순서를 고려하여 데이터를 가지고 있는 자료형이다.

``` python
# 리스트 선언
data = [1, 2, 'Alice', 3]

print(data[1])
print(data[2])

# 리스트에 새로운 요소 추가
data.append('Bob')

print(data)
```

``` bash
[출력]
2
Alice
[1, 2, 'Alice', 3, 'Bob']
```

#### 딕셔너리(Dictionary)

키(Key)-값(Value) 쌍으로 구성된 데이터를 가지고 있는 자료형이다.

``` python
# 딕셔너리 선언
user = {
    'name': 'Alice',
    'age': 21
}

print(user['name'])
print(user['age'])

# 딕셔너리에 새로운 키-값 추가
user['height'] = 160

print(user)
```

``` bash
[출력]
Alice
21
{'name': 'Alice', 'age': 21, 'height': 160}
```
