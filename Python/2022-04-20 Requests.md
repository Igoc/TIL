## Requests

Python으로 HTTP 요청을 할 수 있게 만들어주는 패키지이다.

### 패키지 설치

``` bash
pip install requests
```

### 예시

``` python
import requests

response = requests.get('https://api.thecatapi.com/v1/images/search')
data = response.json()

print(data[0]['url'])
```
