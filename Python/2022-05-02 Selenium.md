## Selenium

웹 브라우저를 이용하는 작업을 자동화하기 위한 도구 모음이다.

### 패키지 설치

``` bash
pip install selenium
```

### 사전 설치

크롬 브라우저를 통해 Selenium을 사용하는 경우, [ChromeDriver](https://chromedriver.chromium.org/)에서 크롬 브라우저와 동일한 버전의 드라이버를 설치해 주어야 한다.

### 스크래핑

Selenium을 통해 스크래핑한 HTML 파일을 Beautiful Soup 패키지로 파싱할 수 있다.

``` python
from bs4 import BeautifulSoup

from selenium import webdriver

from time import sleep

# 크롬 드라이버 실행
driver = webdriver.Chrome('<ChromeDriver 경로>')

# 크롬 드라이버를 통해 특정 URL의 웹 페이지를 띄우기
driver.get('<URL>')

# 동적 웹 페이지가 로딩 완료될 때까지 일정 시간 대기
sleep(5)

# 웹 페이지의 HTML 가져오기
data = driver.page_source()

# 크롬 드라이버 종료
driver.quit()

# Beautiful Soup를 이용한 파싱
soup = BeautifulSoup(data, 'html.parser')
```

### 자바스크립트 실행

`execute_script()`를 이용하여 자바스크립트를 실행할 수 있다.

``` python
from selenium import webdriver

from time import sleep

driver = webdriver.Chrome('<ChromeDriver 경로>')

driver.get('<URL>')
driver.execute_script('<실행할 자바스크립트>')
sleep(5)

data = driver.page_source()
driver.quit()
```

#### 화면의 끝까지 스크롤 내리기

``` python
from selenium import webdriver

from time import sleep

driver = webdriver.Chrome('<ChromeDriver 경로>')

driver.get('<URL>')
driver.execute_script('window.scrollTo(0, document.body.scrollHeight)')
sleep(5)

data = driver.page_source()
driver.quit()
```

### Selector를 이용하여 특정 요소 가져오기

`find_element_by_css_selector()`를 이용하여 Selector로 특정 요소를 가져올 수 있다.

``` python
from selenium import webdriver

from time import sleep

driver = webdriver.Chrome('<ChromeDriver 경로>')

driver.get('<URL>')
element = driver.find_element_by_css_selector('<Selector>')
sleep(5)

data = driver.page_source()
driver.quit()
```

### 특정 요소 클릭하기

`click()`을 이용하여 특정 요소를 클릭할 수 있다.

``` python
from selenium import webdriver

from time import sleep

driver = webdriver.Chrome('<ChromeDriver 경로>')

driver.get('<URL>')
element = driver.find_element_by_css_selector('<Selector>')
element.click()
sleep(5)

data = driver.page_source()
driver.quit()
```
