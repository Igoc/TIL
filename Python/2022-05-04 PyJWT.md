## PyJWT

JSON Web Token을 인코딩 및 디코딩할 수 있는 기능을 제공해주는 Python 패키지이다.

### 패키지 설치

``` bash
pip install PyJWT
```

### JWT 인코딩

``` python
import jwt

SECRET_KEY = '<비밀 키>'

payload = {'<키>': '<값>'}
token = jwt.encode(payload, SECRET_KEY, algorithm='HS256')
```

### JWT 디코딩

``` python
import jwt

SECRET_KEY = '<비밀 키>'

token = '<JWT>'
payload = jwt.decode(token, SECRET_KEY, algorithms='HS256')
```

### 등록된 클레임

#### 만료 시간(Expiration Time)

``` python
from datetime import datetime, timedelta

import jwt
import logging

SECRET_KEY = '<비밀 키>'

# 60초의 유효 시간을 갖는 JWT
payload = {
    '<키>': '<값>',
    'exp': datetime.utcnow() + timedelta(seconds=60)
}

token = jwt.encode(payload, SECRET_KEY, algorithm='HS256')

try:
    payload = jwt.decode(token, SECRET_KEY, algorithms='HS256')
except jwt.ExpiredSignatureError:
    logging.error('JWT 만료')
```

#### 활성화 시간(Not Before Time)

``` python
from datetime import datetime, timedelta

import jwt

SECRET_KEY = '<비밀 키>'

# 60초 후부터 활성화되는 JWT
payload = {
    '<키>': '<값>',
    'nbf': datetime.utcnow() + timedelta(seconds=60)
}

token = jwt.encode(payload, SECRET_KEY, algorithm='HS256')
```

#### 발행자(Issuer)

``` python
import jwt

SECRET_KEY = '<비밀 키>'

# 발행자가 설정된 JWT
payload = {
    '<키>': '<값>',
    'iss': '<JWT 발행자>'
}

token = jwt.encode(payload, SECRET_KEY, algorithm='HS256')

try:
    payload = jwt.decode(token, SECRET_KEY, issuer='<JWT 발행자>', algorithms='HS256')
except jwt.InvalidIssuerError:
    logging.error('올바르지 않은 JWT 발행자')
```

#### 대상자(Audience)

``` python
import jwt

SECRET_KEY = '<비밀 키>'

# 대상자가 설정된 JWT
payload = {
    '<키>': '<값>',
    'aud': '<JWT 대상자>'
}

token = jwt.encode(payload, SECRET_KEY, algorithm='HS256')

try:
    payload = jwt.decode(token, SECRET_KEY, audience='<JWT 대상자>', algorithms='HS256')
except jwt.InvalidAudienceError:
    logging.error('올바르지 않은 JWT 대상자')
```

#### 발행 시간(Issued At)

``` python
import jwt

SECRET_KEY = '<비밀 키>'

# 현재 시간에 발행된 JWT
payload = {
    '<키>': '<값>',
    'iat': datetime.utcnow()
}

token = jwt.encode(payload, SECRET_KEY, algorithm='HS256')
```
