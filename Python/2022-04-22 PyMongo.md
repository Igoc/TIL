## PyMongo

NoSQL 데이터베이스인 MongoDB를 Python에서 조작할 수 있도록 해주는 패키지이다.

### 패키지 설치

``` bash
pip install pymongo
```

### 템플릿

``` python
from pymongo import MongoClient

# localhost:27017에 해당하는 MongoDB에 접속
client = MongoClient('localhost', 27017)

# test 데이터베이스에 연결
db = client.test
```

#### 사용자 정보를 이용한 MongoDB 접속

``` python
from pymongo import MongoClient

# 사용자 이름이 user이고 비밀번호가 password인 계정으로 localhost:27017에 해당하는 MongoDB에 접속
client = MongoClient('mongodb://user:password@localhost', 27017)

# test 데이터베이스에 연결
db = client.test
```

### 도큐먼트 삽입(Insert)

``` python
from pymongo import MongoClient

client = MongoClient('localhost', 27017)
db = client.test

# users 컬렉션에 name이 bobby이고 age가 21인 도큐먼트 삽입
db.users.insert_one({
    'name': 'bobby',
    'age': 21
})
```

### 도큐먼트 조회(Find)

``` python
from pymongo import MongoClient

client = MongoClient('localhost', 27017)
db = client.test

# users 컬렉션에서 name이 bobby인 도큐먼트 중 한 개만 조회
user = db.users.find_one({'name': 'bobby'})

# users 컬렉션의 모든 도큐먼트 조회
users = list(db.users.find({}))

# users 컬렉션에서 age가 21인 모든 도큐먼트를 조회하되, _id 열의 값은 제외하여 반환
users = list(db.users.find({'age': 21}, {'_id': False}))
```

### 도큐먼트 변경(Update)

``` python
from pymongo import MongoClient

client = MongoClient('localhost', 27017)
db = client.test

# users 컬렉션에서 name이 bobby인 도큐먼트 중 한 개만 age를 19로 변경
db.users.update_one({'name': 'bobby'}, {'$set': {'age': 19}})

# users 컬렉션에서 age가 21인 모든 도큐먼트의 age를 19로 변경
db.users.update_many({'age': 21}, {'$set': {'age': 19}})

# posts 컬렉션에서 index가 0인 도큐먼트의 count를 1만큼 증가
db.posts.update_one({'index': 0}, {'$inc': {'count': 1}})
```

### 도큐먼트 삭제(Delete)

``` python
from pymongo import MongoClient

client = MongoClient('localhost', 27017)
db = client.test

# users 컬렉션에서 name이 bobby인 도큐먼트 중 한 개만 삭제
db.users.delete_one({'name': 'bobby'})

# users 컬렉션의 모든 도큐먼트 삭제
db.users.delete_many({})
```

### 도큐먼트 개수 카운트

``` python
from pymongo import MongoClient

client = MongoClient('localhost', 27017)
db = client.test

# users 컬렉션의 도큐먼트 개수 카운트
count = db.users.estimated_document_count({})

# users 컬렉션에서 age가 20인 도큐먼트 개수 카운트
count = db.users.count_documents({'age': 20})
```

### 도큐먼트 정렬

``` python
from pymongo import MongoClient

client = MongoClient('localhost', 27017)
db = client.test

# date를 기준으로 posts 컬렉션의 도큐먼트를 오름차순으로 정렬
posts = list(db.posts.find({}, {'_id': False}).sort('date'))

# date를 기준으로 posts 컬렉션의 도큐먼트를 내림차순으로 정렬
posts = list(db.posts.find({}, {'_id': False}).sort('date', -1))
```
