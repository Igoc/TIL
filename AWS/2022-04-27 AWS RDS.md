## AWS RDS

AWS 클라우드에서 관계형 데이터베이스를 더 쉽게 설치, 운영, 확장할 수 있는 웹 서비스이다.

### 관계형 데이터베이스(Relational Database)

데이터를 열과 행으로 이루어진 테이블에 저장하고, 테이블 사이의 관계를 정의하는 방식으로 데이터 집합을 관리하는 데이터베이스이다.

### 비용 계산

[AWS Pricing Calculator RDS for MySQL](https://calculator.aws/#/createCalculator/RDSMySQL)을 통해 MySQL 엔진을 사용하는 AWS RDS에 대한 대략적인 비용을 산출해볼 수 있다.
