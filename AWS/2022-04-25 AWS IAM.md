## AWS IAM

AWS 리소스에 대한 액세스를 안전하게 제어할 수 있는 웹 서비스이다.

### AWS 액세스 유형

#### 프로그래밍 방식 액세스

AWS API, CLI, SDK 등을 통해 AWS 리소스에 접근할 수 있도록 액세스 키 ID와 비밀 액세스 키를 발급하는 방식이다.

#### AWS Management Console 액세스

AWS Management Console에서 로그인하여 사용할 수 있도록 비밀번호를 발급하는 방식이다.

### 권한 설정

그룹에 사용자 추가, 기존 사용자에서 권한 복사, 기존 정책에 직접 연결을 통해 권한을 부여할 수 있다. 이 때 기존 정책에 직접 연결을 이용하면, 각각의 AWS 리소스를 위해 미리 만들어둔 템플릿을 사용하여 손쉽게 권한을 설정할 수 있다.
