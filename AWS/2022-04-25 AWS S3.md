## AWS S3

AWS에서 제공하는 온라인 스토리지 웹 서비스이다.

### 버킷(Bucket)

AWS S3에서 데이터를 담기 위한 컨테이너를 버킷이라고 부르며, 각각의 버킷에 담기는 데이터는 객체라고 부른다.

### Python과 연동

#### 패키지 설치

``` bash
pip install boto3
```

#### Flask API 구현 예제

``` python
from flask import Flask, jsonify, render_template, request

import boto3

app = Flask(__name__)

# AWS S3에 접속
client = boto3.client(
    service_name='s3',
    aws_access_key_id='<AWS IAM 액세스 키 ID>',
    aws_secret_access_key='<AWS IAM 비밀 액세스 키>',
    region_name='<AWS 리전>'
)

@app.route('/')
def index():
    return render_template('index.html')

@app.route('/upload', methods=['POST'])
def upload():
    file = request.files['file']

    # 모든 사람이 읽을 수 있는 권한을 부여하여 파일을 업로드
    client.put_object(
        Bucket='<버킷명>',
        ACL='public-read',
        Key=file.filename,
        Body=file,
        ContentType=file.content_type
    )

    return jsonify({'status': 'success'})

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

### 정적 호스팅

AWS S3는 단순히 파일을 저장하는 기능뿐만 아니라, HTML과 CSS 등과 같은 정적 파일을 호스팅 할 수 있는 기능을 포함하고 있다. 해당 기능을 이용하기 위해서는 버킷의 속성에 있는 정적 웹 사이트 호스팅을 활성화시켜주면 된다.
