## AWS EC2

AWS에서 제공하는 클라우드 컴퓨팅 서비스이다.

### SSH 접속

``` bash
ssh -i <키 페어> ubuntu@<AWS 인스턴스 퍼블릭 IP>
```

### 서버 환경 설정

#### 한국 시간대로 변경

``` bash
sudo ln -sf /usr/share/zoneinfo/Asia/Seoul /etc/localtime
```

#### pip3 설치

``` bash
sudo apt-get update
sudo apt-get install -y python3-pip
```

#### python3 명령어를 python 명령어로 변경

``` bash
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3 1
```

#### pip3 명령어를 pip 명령어로 변경

``` bash
sudo update-alternatives --install /usr/bin/pip pip /usr/bin/pip3 1
```

#### 포트 포워딩

``` bash
sudo iptables -t nat -A PREROUTING -i eth0 -p tcp --dport <외부 포트> -j REDIRECT --to-port <내부 포트>
```

### nohup

리눅스 OS에서 프로세스를 실행한 터미널의 세션 연결이 끊어지더라도, 프로세스가 지속적으로 작동할 수 있게 만들어주는 명령어이다.

``` bash
nohup <실행할 프로세스> &
```
