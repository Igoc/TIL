## AWS CLI

CLI 환경에서 AWS 서비스를 관리하기 위한 통합 도구로, [AWS 명령줄 인터페이스](https://aws.amazon.com/ko/cli/)에서 자신의 OS에 맞는 AWS CLI를 다운받을 수 있다.

### 버전 확인

``` bash
aws --version
```

### AWS 연결 정보 설정

AWS IAM에서 발급받은 액세스 키 ID와 비밀 액세스 키를 이용하여 연결 정보를 설정한다.

``` bash
aws configure
```

### AWS S3에 파일 업로드

``` bash
# 모든 사람이 읽을 수 있는 권한을 부여하여 파일을 업로드
aws s3 cp <로컬 파일명> s3://<버킷명> --acl public-read
```
