## AWS

AWS는 아마존에서 만든 클라우드 서비스로, Amazon Web Service의 약자이다.

### 리전(Region)

AWS는 여러 국가에 데이터 센터를 구축하고 있으며, 데이터 센터가 위치한 각각의 국가를 리전이라고 부른다. 일반적으로 서비스하려는 지역과 가까운 곳의 리전을 사용할수록 사용자들에게 좀 더 빠른 속도로 서비스를 제공할 수 있기 때문에, AWS를 사용하여 인프라를 구축할 때 서비스 지역에 따라 적절한 리전을 선택해야 한다.

### 존(Zone)

리전 내에서 실제로 데이터 센터가 위치하고 있는 곳을 존이라고 부르며, 하나의 리전은 여러 개의 존을 포함하고 있다. 따라서 특정 존에 있는 데이터 센터에 문제가 발생하더라도, 해당 리전에서 차질 없이 서비스를 이어나가는 것이 가능하다.
