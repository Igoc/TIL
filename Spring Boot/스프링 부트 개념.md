## 스프링 부트란?

- 스프링을 편리하게 사용할 수 있도록 지원해주는 도구이다.
- 단독으로 실행할 수 있는 스프링 애플리케이션을 쉽게 생성할 수 있도록 해준다.
- Tomcat 등과 같은 웹 서버를 내장하여 별도의 웹 서버를 구축하지 않아도 된다.
- 손쉬운 빌드 구성을 위해 starter 종속성을 제공한다.
- 스프링 버전에 알맞은 서드파티 라이브러리 버전을 찾아 자동으로 구성해준다.
- 메트릭, 상태 확인, 외부 구성과 같은 프로덕션 준비 기능을 제공한다.
- 관례에 의한 간결한 설정을 제공해주며, 필요한 부분에 대해서만 별도로 설정하면 된다.
