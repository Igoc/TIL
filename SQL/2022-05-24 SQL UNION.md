## SQL UNION

둘 이상의 `SELECT` 결과를 결합하기 위해 사용되는 연산자이다.

``` sql
/* customers 테이블에서 가져온 city 데이터와 suppliers 테이블에서 가져온 city 데이터를 합치되, 중복된 데이터는 제거하여 결합 */
SELECT city FROM customers
UNION
SELECT city FROM suppliers;

/* customers 테이블에서 가져온 city 데이터와 suppliers 테이블에서 가져온 city 데이터를 합치되, 중복된 데이터도 허용하여 결합 */
SELECT city FROM customers
UNION ALL
SELECT city FROM suppliers;
```
