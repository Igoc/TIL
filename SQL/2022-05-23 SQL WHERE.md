## SQL WHERE

테이블에서 조건에 부합하는 행의 데이터만 가져오기 위해 사용하는 절이다.

``` sql
/* orders 테이블에서 payment_method가 card인 행의 데이터만 가져오기 */
SELECT * FROM orders WHERE payment_method = "card";

/* orders 테이블에서 payment_method가 card이면서 payment_amount가 5000 이상인 행의 데이터만 가져오기 */
SELECT * FROM orders WHERE payment_metohd = "card" AND payment_amount >= 5000;

/* orders 테이블에서 food_name이 제육볶음이 아닌 행의 데이터만 가져오기 */
SELECT * FROM orders WHERE food_name != "제육볶음";
```
