## SQL CASE

조건에 따라서 값을 지정해 주기 위해 사용되는 문장이다.

``` sql
/* points 테이블에서 point 값이 10000 이상이면 VVIP, point 값이 5000 이상이면 VIP, 그렇지 않으면 일반이라고 나타내기 */
SELECT user_id, user_point, (
    CASE
        WHEN user_point >= 10000 THEN "VVIP"
        WHEN user_point >= 5000 THEN "VIP"
        ELSE "일반"
    END
) AS user_level FROM points;
```
