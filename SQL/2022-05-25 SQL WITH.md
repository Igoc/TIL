## SQL WITH

하나의 쿼리문 안에 있는 Subquery를 묶어서 임시 테이블처럼 쓰기 위해 사용하는 절이다.

``` sql
/* check_in 테이블에서 course_id 별로 몇 명의 사용자가 체크인했는지와 orders 테이블에서 course_id 별로 몇 개의 주문이 있는지에 각각 table1과 table2라는 이름을 부여 */
WITH table1 AS (
    SELECT course_id, COUNT(DISTINCT user_id) AS check_in_cnt
    FROM check_in
    GROUP BY course_id
), table2 AS (
    SELECT course_id, COUNT(*) AS order_cnt
    FROM orders
    GROUP BY course_id
);

/* table1과 table2를 INNER JOIN */
SELECT a.course_id, a.check_in_cnt, b.order_cnt
FROM table1 AS a INNER JOIN table2 AS b ON a.course_id = b.course_id;
```
