## SQL AS

테이블 또는 특정 열에 임시적인 이름을 부여하기 위해 사용되는 키워드이다.

``` sql
/* customers 테이블에서 customer_id에 ID, customer_name에 customer라는 이름을 부여하여 데이터 가져오기 */
SELECT customer_id AS ID, customer_name AS customer FROM customers;

/* customers 테이블에 c, orders 테이블에 o라는 이름을 부여하여 INNER JOIN 한 후, customers 테이블에 있는 customer_name과 orders 테이블에 있는 order_date 데이터 가져오기 */
SELECT c.customer_name, o.order_date FROM customers AS c INNER JOIN orders AS o ON c.customer_id = o.customer_id;
```
