## Java

### 변수

어떤 데이터를 저장하기 위한 메모리 공간에 이름을 붙여둔 것으로, 언제든지 할당한 값을 변경할 수 있다.

``` java
// int 자료형 변수 선언
int num = 5;

System.out.println(num);

// String 자료형 변수 선언
String str = "Hello, world!";

System.out.println(str);

// 변수에 할당한 값 변경
str = "Bye, world!";

System.out.println(str);
```

``` bash
[출력]
5
Hello, world!
Bye, world!
```

### 상수

변수와 달리 초기에 할당한 값을 나중에 변경할 수 없다.

``` java
final int MAX = 10;

System.out.println(MAX);
```

``` bash
[출력]
10
```

### 자료형

#### 기본 자료형(Primitive Data Type)

##### 숫자 자료형

``` java
// 정수형 변수 선언
byte num1 = 'A';
short num2 = 1;
int num3 = 3;
long num4 = 1234567890L;

System.out.println(num1);
System.out.println(num2);
System.out.println(num3);
System.out.println(num4);

// 실수형 변수 선언
float num5 = 5.5F;
double num6 = 3.14;

System.out.println(num5);
System.out.println(num6);
```

``` bash
[출력]
65
1
3
1234567890
5.5
3.14
```

#### 참조 자료형(Reference Data Type)

Java에서는 기본 자료형을 제외한 모든 자료형을 참조 자료형으로 취급한다.

``` java
// String 자료형 변수 선언
String name = "Alice";

System.out.println(name);

// int 배열 자료형 변수 선언
int[] array = new int[] {1, 2, 3};

System.out.println(Arrays.toString(array));
```

``` bash
[출력]
Alice
[1, 2, 3]
```

##### 배열

동일한 자료형의 데이터를 연속된 공간에 저장하기 위한 자료 구조이다.

``` java
// int 배열 자료형 변수 선언
int[] array1 = new int[5];

System.out.println(Arrays.toString(array1));

// int 배열 자료형 변수를 선언과 동시에 초기화
int[] array2 = new int[] {1, 2, 3, 4, 5};

System.out.println(Arrays.toString(array2));

// String 배열 자료형 변수 선언
String[] array3 = new String[5];

System.out.println(Arrays.toString(array3));

// String 배열 자료형 변수를 선언과 동시에 초기화
String[] array4 = {"Alice", "Bob", "Chris"};

System.out.println(Arrays.toString(array4));

// 인덱스를 통해 배열 값에 접근
int[] array5 = new int[3];
array5[0] = 5;
array5[1] = 10;

System.out.println(array5[1]);

// 배열의 길이
int[] array6 = new int[10];

System.out.println(array6.length);
```

``` bash
[출력]
[0, 0, 0, 0, 0]
[1, 2, 3, 4, 5]
[null, null, null, null, null]
[Alice, Bob, Chris]
10
10
```

### 연산자

#### 산술 연산자

``` java
int num1 = 10;
int num2 = 5;

System.out.println(num1 + num2); // 더하기 연산
System.out.println(num1 - num2); // 빼기 연산
System.out.println(num1 * num2); // 곱하기 연산
System.out.println(num1 / num2); // 나누기 연산
System.out.println(num1 % num2); // 나머지 연산
```

``` bash
[출력]
15
5
50
2
0
```

#### 대입 연산자

``` java
int num1 = 10;
int num2 = 5;

// num1 = num1 + num2
num1 += num2;

System.out.println(num1);

// num1 = num1 - num2
num1 -= num2;

System.out.println(num1);

// num1 = num1 * num2
num1 *= num2;

System.out.println(num1);

// num1 = num1 / num2
num1 /= num2;

System.out.println(num1);

// num1 = num1 % num2
num1 %= num2;

System.out.println(num1);
```

``` bash
[출력]
15
10
50
10
0
```

#### 관계 연산자

``` java
int num1 = 10;
int num2 = 20;
int num3 = 10;

System.out.println(num1 > num2); // 10 > 20
System.out.println(num1 >= num3); // 10 >= 10
System.out.println(num1 < num2); // 10 < 20
System.out.println(num1 <= num2); // 10 <= 20
System.out.println(num1 == num3); // 10 == 10
System.out.println(num1 != num2); // 10 != 20
```

``` bash
[출력]
false
true
true
true
true
true
```

#### 논리 연산자

``` java
boolean a = true;
boolean b = false;

System.out.println(a && b); // 논리 곱 연산
System.out.println(a || b); // 논리 합 연산
System.out.println(!b); // 논리 부정 연산
```

``` bash
[출력]
false
true
true
```
