## Java

### 변수

어떤 데이터를 저장하기 위한 메모리 공간에 이름을 붙여둔 것으로, 언제든지 할당한 값을 변경할 수 있다.

``` java
// int 자료형 변수 선언
int num = 5;

System.out.println(num);

// String 자료형 변수 선언
String str = "Hello, world!";

System.out.println(str);

// 변수에 할당한 값 변경
str = "Bye, world!";

System.out.println(str);
```

``` bash
[출력]
5
Hello, world!
Bye, world!
```

### 상수

변수와 달리 초기에 할당한 값을 나중에 변경할 수 없다.

``` java
final int MAX = 10;

System.out.println(MAX);
```

``` bash
[출력]
10
```

##### 배열

동일한 자료형의 데이터를 연속된 공간에 저장하기 위한 자료 구조이다.

``` java
// int 배열 자료형 변수 선언
int[] array1 = new int[5];

System.out.println(Arrays.toString(array1));

// int 배열 자료형 변수를 선언과 동시에 초기화
int[] array2 = new int[] {1, 2, 3, 4, 5};

System.out.println(Arrays.toString(array2));

// String 배열 자료형 변수 선언
String[] array3 = new String[5];

System.out.println(Arrays.toString(array3));

// String 배열 자료형 변수를 선언과 동시에 초기화
String[] array4 = {"Alice", "Bob", "Chris"};

System.out.println(Arrays.toString(array4));

// 인덱스를 통해 배열 값에 접근
int[] array5 = new int[3];
array5[0] = 5;
array5[1] = 10;

System.out.println(array5[1]);

// 배열의 길이
int[] array6 = new int[10];

System.out.println(array6.length);
```

``` bash
[출력]
[0, 0, 0, 0, 0]
[1, 2, 3, 4, 5]
[null, null, null, null, null]
[Alice, Bob, Chris]
10
10
```

### 연산자

#### 산술 연산자

``` java
int num1 = 10;
int num2 = 5;

System.out.println(num1 + num2); // 더하기 연산
System.out.println(num1 - num2); // 빼기 연산
System.out.println(num1 * num2); // 곱하기 연산
System.out.println(num1 / num2); // 나누기 연산
System.out.println(num1 % num2); // 나머지 연산
```

``` bash
[출력]
15
5
50
2
0
```

#### 대입 연산자

``` java
int num1 = 10;
int num2 = 5;

// num1 = num1 + num2
num1 += num2;

System.out.println(num1);

// num1 = num1 - num2
num1 -= num2;

System.out.println(num1);

// num1 = num1 * num2
num1 *= num2;

System.out.println(num1);

// num1 = num1 / num2
num1 /= num2;

System.out.println(num1);

// num1 = num1 % num2
num1 %= num2;

System.out.println(num1);
```

``` bash
[출력]
15
10
50
10
0
```

#### 관계 연산자

``` java
int num1 = 10;

int num3 = 10;

System.out.println(num1 >= num3); // 10 >= 10

System.out.println(num1 == num3); // 10 == 10
System.out.println(num1 != num2); // 10 != 20
```

``` bash
[출력]

true
```

#### 삼항 연산자

`?` 앞의 조건식이 참이면 `:` 앞의 문장을 실행하고, 거짓이라면 `:` 뒤의 문장을 실행하는 연산자이다.

``` java
int num = 5;
String result = (num < 10) ? ("10 미만") : ("10 이상");

System.out.println(result);
```

``` bash
[출력]
10 미만
```

### 조건문

조건의 결과에 따라 프로그램 실행 흐름을 분기하기 위한 구문이다.

#### if-else 문

`if` 조건식이 참이라면 `if` 뒤의 명령문을 실행하고, 거짓이라면 `else` 뒤의 명령문을 실행한다.

``` java
int age = 20;

if (age < 10)
    System.out.println("10살 미만");
else if (age < 20) {
    System.out.println("20살 미만");
}
else {
    if (age == 20) {
        System.out.println("20살");
    }
    else if (age == 21) {
        System.out.println("21살");
    }
    else {
        System.out.println("22살 이상");
    }
}
```

``` bash
[출력]
20살
```

#### switch 문

`switch` 뒤에 오는 변수의 값에 따라 실행을 분기하는 구문으로, 변수의 값과 일치하는 `case` 뒤에 오는 모든 명령문을 실행한다.

``` java
int dayOfWeek = 0;

switch (dayOfWeek) {
    case 0:
        System.out.println("월요일");

    case 1:
        System.out.println("화요일");

    case 2:
        System.out.println("수요일");
        break;

    case 3:
        System.out.println("목요일");
        break;

    case 4:
        System.out.println("금요일");
        break;

    case 5:
        System.out.println("토요일");
        break;

    case 6:
        System.out.println("일요일");
        break;

    default:
        System.out.println("오류");
}
```

``` bash
[출력]
월요일
화요일
수요일
```

### 반복문

특정 명령문을 여러 차례 반복하여 실행하기 위한 구문이다.

#### for 문

초기식, 조건식, 증감식 구조로 구성되며, 조건식을 만족하는 동안 뒤에 오는 명령문을 반복하여 실행한다.

``` java
for (int index = 0; index < 3; index++) {
    System.out.println(index);
}
```

``` bash
[출력]
0
1
2
```

#### for-each 문

배열 또는 `java.lang.Iterable`을 상속받은 클래스의 요소를 순차적으로 하나씩 순회한다.

``` java
int[] array = { 1, 2, 3 };

for (int x : array) {
    System.out.println(x);
}
```

``` bash
[출력]
1
2
3
```

#### while 문

조건식이 참인 동안 뒤에 오는 명령문을 반복해서 실행한다.

``` java
int index = 0;

while (index < 3) {
    System.out.println(index);
    index++;
}
```

``` bash
[출력]
0
1
2
```

#### do-while 문

while 문처럼 조건식이 참인 동안 `do` 뒤에 오는 명령문을 반복하여 실행하나, 조건의 만족 여부와 상관없이 무조건 `do` 뒤에 오는 명령문을 1회 수행한다.

``` java
int index = 3;

do {
    System.out.println(index);
    index++;
} while(index < 3);
```

``` bash
[출력]
3
```

#### 반복문 제어

반복문의 뒤에 오는 명령문을 수행하는 도중 반복을 제어하기 위한 구문이다.

##### continue

명령문 수행 도중 `continue`를 만나면, 뒤에 오는 명령문을 건너뛰고 다음 반복을 수행한다.

``` java
for (int index = 0; index < 5; index++) {
    if (index == 2) {
        continue;
    }

    System.out.println(index);
}
```

``` bash
[출력]
0
1
3
4
```

##### break

명령문 수행 도중 `break`를 만나면, 뒤에 오는 명령문을 건너뛰고 반복을 즉시 종료한다.

``` java
for (int index = 0; index < 5; index++) {
    if (index == 2) {
        break;
    }

    System.out.println(index);
}
```

``` bash
[출력]
0
1
```

### 클래스(Class)

어떤 대상이 가질 수 있는 상태와 행위의 집합을 정의해놓은 것으로, 클래스에서 상태와 행위는 각각 필드와 메서드라는 형태로 표현된다.

``` java
class Car {
    private int speed = 0;

    public void go() {
        speed += 10;
    }

    public void back() {
        speed -= 10;
    }
}
```

#### 인스턴스(Instance)

클래스를 사용하기 위해 메모리 상에 할당한 것을 인스턴스라고 부른다.

``` java
Car car1 = new Car();
Car car2 = new Car();
```

#### 생성자(Constructor)

인스턴스가 생성될 때 사용되는 인스턴스 초기화 메서드로, 해당 클래스의 인스턴스가 메모리에 할당되는 시점에 자동으로 호출된다. 만약 클래스 내에 아무런 생성자도 정의되어 있지 않다면, Java 컴파일러에서 아무런 작업도 하지 않는 기본 생성자를 자동으로 생성해준다.

``` java
class Phone {
    private String model;
    private String color;

    Phone(String model, String color) {
        this.model = model;
        this.color = color;
    }
}

class Main {
    public static void main(String[] args) {
        Phone galaxy = new Phone("Galaxy S10", "Black");
    }
}
```

### 상속(Inheritance)

재사용 또는 확장을 위해 어떤 클래스나 인터페이스의 필드와 메서드를 물려받을 수 있는 기법이다.

#### 클래스 상속

`extends`를 이용하여 슈퍼 클래스에 정의된 필드와 메서드를 서브 클래스에서 물려받을 수 있으며, Java에서는 클래스 다중 상속을 허용하지 않는다.

``` java
class Animal {
    protected String name;

    public void cry() {
        System.out.println(name + " 울음");
    }
}

class Dog extends Animal {
    Dog(String name) {
        this.name = name;
    }

    public void swim() {
        System.out.println(name + " 수영");
    }
}

class Main {
    public static void main(String[] args) {
        Dog dog1 = new Dog("코코");
        dog1.cry();
        dog1.swim();

        Animal dog2 = new Dog("초코");
        dog2.cry();
    }
}
```

``` bash
[출력]
코코 울음
코코 수영
초코 울음
```

##### super

서브 클래스에서 슈퍼 클래스에 정의된 필드나 메서드를 사용하기 위한 키워드이다.

``` java
class Super {
    protected int num = 10;
}

class Sub extends Super {
    protected int num = 20;

    public void display() {
        System.out.println(num);
        System.out.println(this.num);
        System.out.println(super.num);
    }
}

class Main {
    public static void main(String[] args) {
        Sub sub = new Sub();
        sub.display();
    }
}
```

``` bash
[출력]
20
20
10
```

### 오버로딩(Overloading)

하나의 클래스 내에서 동일한 이름의 메서드를 여러 개 정의하는 것으로, 오버로딩된 메서드들이 갖는 파라미터의 개수나 자료형이 반드시 달라야 한다.

``` java
class Adder {
    public int add(int x, int y) {
        return x + y;
    }

    public int add(int x, int y, int z) {
        return x + y + z;
    }

    public float add(float x, float y) {
        return x + y;
    }
}
```

### 오버라이딩(Overriding)

슈퍼 클래스에 정의된 메서드를 서브 클래스에서 재정의하는 것으로, 오버라이딩된 메서드의 시그니처는 슈퍼 클래스에서 정의한 것과 동일해야 한다.

``` java
class Animal {
    public void cry() {
        System.out.println("울음");
    }
}

class Dog extends Animal {
    public void cry() {
        System.out.println("짖음");
    }
}

class Main {
    public static void main(String[] args) {
        Animal dog = new Dog();
        dog.cry();
    }
}
```

``` bash
[출력]
짖음
```

### 패키지(Package)

서로 관련성을 갖는 클래스나 인터페이스를 디렉터리 단위로 묶어 놓은 것을 의미한다.

#### 접근 제어자(Access Modifier)

외부에서 클래스에 정의한 필드나 메서드에 접근하는 것을 제한하는 역할을 수행한다.

| 접근 제어자 | 같은 패키지의 클래스 | 같은 패키지의 서브 클래스 | 다른 패키지의 서브 클래스 | 다른 패키지의 클래스 |
| --- | --- | --- | --- | --- |
| public | O | O | O | O |
| protected | O | O | O | X |
| default | O | O | X | X |
| private | X | X | X | X |
