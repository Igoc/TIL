## Flask

Python으로 작성된 마이크로 웹 프레임워크 패키지이다.

### 패키지 설치

``` bash
pip install Flask
```

### 시작하기

``` python
from flask import Flask

app = Flask(__name__)

@app.route('/')
def index():
    return 'Hello, world!'

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

### route 데코레이터

특정 URL에 대한 요청이 들어왔을 때 실행될 함수를 설정하기 위한 데코레이터이다.

``` python
@app.route('/page')
def page():
    return 'This is page!'
```

### 프로젝트 구조

#### app.py

Flask 애플리케이션에 대한 코드를 작성하는 파일이다.

#### templates 폴더

렌더링 될 HTML 파일을 저장하는 폴더이다.

#### static 폴더

이미지, CSS 파일 등을 저장하는 폴더이다.

### HTML 파일 렌더링

render_template() 함수를 이용하여 templates 폴더에 저장한 HTML 파일을 렌더링 할 수 있다.

``` python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    # index.html 렌더링
    return render_template('index.html')

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

### Jinja를 이용한 HTML 파일 렌더링

``` python
from flask import Flask, render_template

app = Flask(__name__)

@app.route('/')
def index():
    user = {
        'name': 'Alice',
        'age': 20
    }

    status = 'new'

    return render_template('index.html', user=user, status=status)

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

### URL 변수

### GET 파라미터 수신

``` python
from flask import Flask, request

app = Flask(__name__)

@app.route('/user', methods=['GET'])
def get_user():
    # name 파라미터 수신
    name = request.args.get('name')

    return name

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

### POST 폼 데이터 수신

``` python
from flask import Flask, request

app = Flask(__name__)

@app.route('/user', methods=['POST'])
def create_user():
    # name 폼 데이터 수신
    name = request.form['name']

    # hobbies 폼 리스트 데이터 수신
    hobbies = request.form.getlist('hobbies[]')

    return name

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

### POST 파일 데이터 수신

``` python
from flask import Flask, request

app = Flask(__name__)

@app.route('/image', methods=['POST'])
def save_image():
    # image 파일 데이터 수신
    image = request.files['image']

    # 파일 저장
    image.save(f'<저장 경로>/{image.filename}')

    return image.filename

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

### Cookie 획득

``` python
from flask import Flask, request

app = Flask(__name__)

@app.route('/')
def index():
    data = request.cookies.get('<Cookie 키>')

    return data

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

#### 크로스 도메인 요청 Cookie 허용

``` python
from flask import Flask

app = Flask(__name__)

@app.after_request
def after_request(response):
    response.headers.add('Access-Control-Allow-Credentials', 'true')

    return response

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```

### 리다이렉트

``` python
from flask import Flask, redirect, request, url_for

app = Flask(__name__)

@app.route('/')
def index():
    msg = request.args.get('msg', 'default')

    return msg

@app.route('/test')
def test():
    return redirect(url_for('index', msg='test'))

if __name__ == '__main__':
    app.run('0.0.0.0', port=5000, debug=True)
```
