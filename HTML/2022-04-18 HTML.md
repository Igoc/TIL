## HTML

HTML은 화면에 그려질 내용의 뼈대를 담당한다.

### HTML 기초

``` html
<!DOCTYPE html>

<html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <title>HTML 기초</title>
    </head>
    <body>
        <!-- 구역을 나누는 태그 -->
        <div>구역 나누기</div>

        <!-- 문단을 만드는 태그 -->
        <p>문단 만들기</p>

        <!-- 목록을 만드는 태그 -->
        <ul>
            <li>목록 1</li>
            <li>목록 2</li>
        </ul>

        <!-- 주제목을 나타내는 태그 -->
        <h1>주제목</h1>

        <!-- 소제목을 나타내는 태그 -->
        <h2>소제목</h2>

        <!-- 구분선을 나타내는 태그 -->
        <hr/>

        <!-- 특정 글자를 꾸미기 위한 태그 -->
        특정 <span style="color: red">글자를</span> 꾸미는 태그

        <!-- 하이퍼링크를 나타내는 태그 -->
        <a href="https://www.google.com/">하이퍼링크</a>

        <!-- 이미지를 나타내는 태그 -->
        <img src="https://www.google.com/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png"/>

        <!-- 텍스트 입력 박스를 나타내는 태그 -->
        <input type="text"/>

        <!-- 버튼을 나타내는 태그 -->
        <button>버튼</button>

        <!-- 텍스트 입력 영역을 나타내는 태그 -->
        <textarea>텍스트 입력 영역</textarea>
    </body>
</html>
```

### a 태그를 통해 자바스크립트 함수 호출

``` html
<a href="javascript:<자바스크립트 함수 호출>">...</a>
```
