## CSS

CSS는 화면을 꾸미는 역할을 담당한다.

### CSS 기초

`<head>...</head>` 태그 안에 `<style>...</style>` 태그로 공간을 만들어 CSS를 작성한다.

### margin과 padding

`margin`은 바깥쪽 여백을 지정하기 위해 사용하고, `padding`은 안쪽 여백을 지정하기 위해 사용한다.

### 이미지 설정하기

CSS로 특정 태그에 이미지를 설정할 때, 보통 다음과 같이 3가지를 함께 사용한다.

``` css
/* 이미지 지정 */
background-image: url('https://www.ancient-origins.net/sites/default/files/field/image/Agesilaus-II-cover.jpg');

/* 이미지 정렬 */
background-position: center;

/* 이미지 크기 */
background-size: cover;
```

### 수평 중앙 정렬

``` css
/* width, margin 설정만으로 안되는 경우 display 변경 필요 */
display: block;

/* 바깥쪽 여백을 지정하기 위해 크기 조절 필요 */
width: 500px;

/* 남는 공간에 균등하게 여백 생성 */
margin: auto;
```

### 구글 웹 폰트

[Google Fonts](https://fonts.google.com/)에서 원하는 폰트를 선택하고, 아래와 같은 형태로 HTML과 CSS를 설정해주면 된다.

``` html
<!DOCTYPE html>

<html>
    <head>
        <link rel="preconnect" href="https://fonts.googleapis.com">
        <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
        <link href="https://fonts.googleapis.com/css2?family=Nanum+Gothic&display=swap" rel="stylesheet">

        <style>
            * {
                font-family: 'Nanum Gothic', sans-serif;
            }
        </style>
    </head>
    <body>
    </body>
</html>
```

### CSS 파일 분리

`<style>...</style>` 태그 안의 내용을 별도의 CSS 파일로 만들고, `<head>...</head>` 태그 안에 아래와 같은 내용을 추가해주면 된다.

``` html
<link rel="stylesheet" type="text/css" href = "<CSS 파일명>.css">
```

### 부트스트랩

트위터에서 사용하는 여러 컴포넌트에 대한 CSS를 미리 정의해놓은 것으로, 컴포넌트 목록은 [Bootstrap Components](https://getbootstrap.com/docs/4.0/components/)에서 확인할 수 있다. 일반적으로 부트스트랩을 사용하기 위해 아래와 같은 템플릿을 이용한다.

``` html
<!DOCTYPE html>

<html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/css/bootstrap.min.css" integrity="sha384-Gn5384xqQ1aoWXA+058RXPxPg6fy4IWvTNh0E263XmFcJlSAwiGgFAW/dAiS6JXm" crossorigin="anonymous">

        <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.9/umd/popper.min.js" integrity="sha384-ApNbgh9B+Y1QKtv3Rn7W3mgPxhU9K/ScQsAP7hUibX39j7fakFPskvXusvfa0b4Q" crossorigin="anonymous"></script>
        <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0/js/bootstrap.min.js" integrity="sha384-JZR6Spejh4U02d8jOt6vLEHfe/JQGiRRSQQxSfFWpi1MquVdAyjUar5+76PVCmYl" crossorigin="anonymous"></script>

        <title>부트스트랩 템플릿</title>
    </head>
    <body>
    </body>
</html>
```
