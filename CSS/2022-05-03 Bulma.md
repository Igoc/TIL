## Bulma

반응형 웹 인터페이스를 구축하기 위해 사용할 수 있는 오픈 소스 CSS 프레임워크로, 컴포넌트 목록은 [Bulma Documentation](https://bulma.io/documentation/)에서 확인할 수 있다.

``` html
<!DOCTYPE html>

<html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">

        <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bulma@0.9.3/css/bulma.min.css">

        <title>Bulma 템플릿</title>
    </head>
    <body>
    </body>
</html>
```
