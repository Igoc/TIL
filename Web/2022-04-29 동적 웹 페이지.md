## 동적 웹 페이지

서버에 저장되어 있는 HTML 파일을 상황에 따라 동적으로 변경하여 클라이언트에게 전송하는 방식의 웹 페이지이다.

### 장점

- 사용자의 요청에 따라 HTML 파일을 동적으로 변화시킬 수 있기 때문에 다양한 서비스를 할 수 있다.

### 단점

- 사용자의 입력을 받기 때문에 상대적으로 보안에 취약하다.
- 검색 엔진 최적화(SEO)가 어렵다.

### 종류

#### 클라이언트 사이드 렌더링(CSR)

클라이언트에서 자바스크립트를 이용하여 콘텐츠들을 렌더링하는 방식이다.

#### 서버 사이드 렌더링(SSR)

서버에서 템플릿 HTML 파일에 데이터를 끼워 넣어 완성된 웹 페이지를 만들고, 이를 클라이언트에게 전송하는 방식이다.
