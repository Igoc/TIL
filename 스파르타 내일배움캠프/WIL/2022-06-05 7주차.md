## 7주차

### 이번 주에 배운 것

- 웹 개발의 봄, Spring 강의를 들으며, Spring을 이용하여 WAS를 구축하는 방법을 배울 수 있었다.
- 스프링 입문 - 코드로 배우는 스프링 부트, 웹 MVC, DB 접근 기술 강의를 들으며, Spring 웹 애플리케이션 개발에 필요한 전반적인 지식을 습득할 수 있었다.
- 서버리스 백엔드 실시간 강의를 들으며, AWS에서 서버리스 백엔드를 구축하는 방법을 익혔다.
- 서버리스 배포 자동화 실시간 강의를 들으며, AWS에 서버리스를 자동으로 배포할 수 있는 방법을 익혔다.
- 타임 어택을 수행하며, AWS VPC에서 제공하는 게이트웨이, 라우팅 테이블, 서브넷의 동작 구조를 이해할 수 있었다.
