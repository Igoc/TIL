## Azure Virtual Network

Azure 리소스의 통신을 위해 가상 네트워크를 구축할 수 있는 서비스이다.

### 보안

#### 네트워크 보안 그룹(Network Security Group)

가상 네트워크에 있는 리소스가 주고받는 네트워크 트래픽을 필터링할 수 있는 기능이다.

##### 특징

- 통신에 사용되는 인바운드/아웃바운드 규칙을 설정할 수 있다.
- 규칙마다 통신을 허용할 것인지 거부할 것인지 여부와 우선순위를 설정할 수 있다.
- Azure에서 지정한 태그를 기반으로 필터링할 수 있다.

#### 애플리케이션 보안 그룹(Application Security Group)

가상 머신을 그룹화하고, 해당 그룹에 따라 네트워크 보안 정책을 정의할 수 있는 기능이다.

##### 특징

- 유사한 정책을 갖는 가상 머신을 그룹화하여 정책의 개수를 간소화할 수 있다.
- IP 주소를 명시적으로 지정하지 않고도, 각각의 가상 머신에 보안 정책을 적용할 수 있다.
