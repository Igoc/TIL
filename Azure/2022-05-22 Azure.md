## Azure

Azure는 마이크로소프트에서 만든 클라우드 서비스이다.

### 인프라 구조

#### 지리(Geographies)

하나 이상의 지역이 포함된 단위로 통상적으로 국가 단위로 구성된다.

#### 지역(Regions)

하나 또는 여러 개로 구성된 데이터 센터가 위치한 지역을 나타낸다.

#### 가용성 영역(Availability Zones)

하나의 지역 내에서 물리적으로 구분된 데이터 센터들을 나타내며, 2개 이상의 가용성 영역에 배포하는 것을 고가용성이라고 부른다.

#### 지역 쌍(Region Pair)

특정 지역에서 자연재해, 침수 등이 발생하여 서비스가 불가능해졌을 때, 해당 지역과 쌍을 이루고 있는 지역을 통해 장애 극복(Failover)을 수행한다.

### 리소스 구조

#### 관리 그룹(Management Groups)

여러 개의 구독을 묶어서 관리하기 위한 그룹이다.

#### 구독(Subscriptions)

Azure가 관리하는 결제 서비스로, 하나의 계정에서 여러 개의 구독을 생성할 수 있다.

#### 리소스 그룹(Resource Groups)

Azure 솔루션에 관련된 리소스들을 묶어서 관리하기 위한 컨테이너이다.

#### 리소스(Resources)

Azure가 관리하는 가상 머신, 가상 네트워크 등의 서비스를 의미한다.
